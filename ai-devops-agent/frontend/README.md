# Frontend (Next.js)

Frontend-only AI DevOps Copilot dashboard using mock data.

## Stack
- Next.js (App Router)
- React + TypeScript (strict)
- Tailwind CSS
- shadcn/ui primitives
- Framer Motion
- Lucide icons
- Sonner toasts

## Run
```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Notes
- No backend/API calls are implemented.
- All UI data comes from `src/lib/mockData.ts`.
- Pages:
  - `/`
  - `/dashboard`
  - `/run/run_1002`
  - `/repo/repo_001`
  - `/history`
